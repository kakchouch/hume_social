# Hume Social Documentation

This folder provides 3 depth levels so different audiences can onboard quickly.

## Documentation Levels

1. [Beginner Guide](beginner.md)
   Audience: Product, PM, new contributors.
   Focus: What the platform does and how to run it.

2. [Intermediate Guide](intermediate.md)
   Audience: Active contributors.
   Focus: Request flows, app boundaries, and implementation patterns.

3. [Expert Reference](expert.md)
   Audience: Maintainers and reviewers.
   Focus: Full inventory of apps, models, views, templates, settings, and operational details.
