Hume Social Documentation

This folder provides 3 depth levels so different audiences can onboard quickly.

Documentation Levels

  1. Beginner Guide Audience: Product, PM, new contributors. Focus: What the platform does and how to run it.

  2. Intermediate Guide Audience: Active contributors. Focus: Request flows, app boundaries, and implementation patterns.

  3. Expert Reference Audience: Maintainers and reviewers. Focus: Full inventory of apps, models, views, templates, settings, and operational details.

  4. Object Reference Map Audience: Developers needing object-by-object navigation. Focus: Linked documentation for classes, parameters, templates, and cross-app relationships.

Suggested Reading Path

  1. Start with Beginner Guide.
  2. Move to Intermediate Guide when you begin coding.
  3. Use Expert Reference as the canonical technical reference.
  4. Use Object Reference Map to navigate directly between linked objects.
